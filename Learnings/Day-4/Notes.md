## Databse Basics
* Database is a collection of structured information(Data). We can see databases in day to day life like employee database in a company, telephone directory etc. Here we store in the format of files.
* Ex: Microsoft acess

* A Relational database is a structured database which is having tables and having realationships between tables or coloumns. Here we store data in the format of tables.
* Ex: Mysql,Oracle,SQLite

## Keys

* primary key : a column or set of columns that uniquely identify each row or record in a table.
* foreign key : a column or set of columns in one table refers to the primary key of another table.
* super key : it is a group of single or multiple keys that uniquely identifies rows in a table.
* candidate key :it is a minimal super key
